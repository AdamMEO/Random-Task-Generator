# Random Task Generator

**Автор:** [Адам Межгихов]

## Описание

«Random Task Generator» — это простое GUI‑приложение на Python, которое помогает выбирать случайные задачи из списка. Программа позволяет:

* генерировать случайные задачи по нажатию кнопки;
* просматривать историю сгенерированных задач;
* фильтровать историю по типам задач (учёба, работа, спорт, отдых);
* добавлять новые задачи в общий список;
* сохранять историю задач между запусками приложения (в формате JSON).

Приложение создано с использованием стандартной библиотеки Python `tkinter` для графического интерфейса и модуля `random` для случайного выбора задач.

## Установка и запуск

### Требования

* Python 3.6 или выше.
